border-image.com demonstrates how it’s sliced.

Non-retina compatibility?

On non-retina screens this doesn’t look good: . I.e. border becomes half-transparent. Fix:

.thin-border {
	border: 1px solid black;
}
@media (-webkit-min-device-pixel-ratio: 2), (min-resolution: 192dpi) {
	.thin-border {
		-webkit-border-image: url(border.gif) 2 repeat;
		border-image: url(border.gif) 2 repeat;
	}
}

Browsers support

I’ve tested in on Chrome 31, Safari 7, Firefox 26, and iOS 7 — all works.
